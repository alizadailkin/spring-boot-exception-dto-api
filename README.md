# Spring Boot Employer-Department API

A Spring Boot REST API project showcasing best practices for exception handling and the use of DTOs to structure data transfer.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- RESTful API
- PostgreSQL / H2 (optional)
- Lombok
- DBeaver (database management & query testing)
- Postman (API testing)
- Git & GitHub

## Features
- Employer and Department CRUD operations
- DTOs for clean data transfer
- Global exception handling
- Layered architecture (Controller, Service, Repository)

## Goal
This project is created to improve my understanding of exception handling, DTO patterns, and structured backend development in real-world scenarios.
